# ABoroOffice – Einfaches Hilfe-Handbuch

## Wie du das liest
- Jede App hat: Was ist das? Wofür ist das? Was muss ich klicken?
- Wenn etwas nicht klappt: Checkliste am Ende.

## 1) CRM (Kunden und Leads)
**Was ist das?**
Hier speicherst du Kunden und neue Interessenten (Leads).

**Wofür ist das?**
- Neue Kontakte sammeln
- Verkaufschancen verfolgen
- Angebot im ERP starten

**So machst du es (ganz einfach)**
1. Lead anlegen (Name und Firma).  
2. Website eintragen (damit automatische Daten kommen).  
3. Notiz schreiben (z. B. „Rückruf nächste Woche“).  
4. Opportunity anlegen und Stage wählen.  
5. Wenn gewonnen -> Angebot im ERP starten.

## 2) ERP (Angebot bis Rechnung)
**Was ist das?**
Hier schreibst du Angebote, Aufträge, Rechnungen und Mahnungen.

**So machst du es**
1. Angebot erstellen.  
2. Angebot annehmen -> Auftrag entsteht.  
3. Auftragsbestätigung wird erstellt.  
4. Auftrag erledigen -> Rechnung erzeugen.  
5. Falls nötig: Mahnung erstellen.

## 3) Projekte (Kanban und Gantt)
**Was ist das?**
Projektplanung mit Aufgaben.

**So machst du es**
1. Projekt anlegen.  
2. Aufgaben erstellen.  
3. Kanban: Aufgaben verschieben.  
4. Gantt: Zeitplan anschauen.

**Kurz erklärt**
- Kanban = Tagesarbeit (was ist offen, was läuft, was fertig).  
- Gantt = Planung (wann passiert was).

## 4) Workflows (Automatik)
**Was ist das?**
Das System macht automatisch etwas, wenn ein Ereignis passiert.

**So machst du es**
1. Workflow erstellen.  
2. Trigger auswählen (CRM/ERP/Marketing).  
3. Aktion auswählen (E-Mail, Task, Kommentar).  
4. Aktivieren.

## 5) Marketing
**Was ist das?**
Hier planst du Inhalte (Posts, Newsletter, etc.).

**So machst du es**
1. Briefing schreiben.  
2. KI-Entwurf erzeugen.  
3. Freigabe anfordern.  
4. Veröffentlichen.

## 6) Verträge
**Was ist das?**
Verträge speichern und prüfen.

**So machst du es**
1. Vertrag anlegen.  
2. Datei hochladen.  
3. KI-Analyse starten.  
4. Versionen verwalten.

## 7) Personal
**Was ist das?**
Mitarbeiter und Dozenten verwalten.

**So machst du es**
1. Abteilung anlegen.  
2. Mitarbeiter/Dozenten hinzufügen.  
3. Zeiterfassung pflegen.  
4. Export für Lohnabrechnung.

## 8) FiBu
**Was ist das?**
Finanzbuchhaltung.

**So machst du es**
1. Kontenplan prüfen/importieren.  
2. Buchungen im Journal.  
3. Auto-Postings aus ERP prüfen.

## 9) Helpdesk
**Was ist das?**
Support-Tickets und Chat.

**So machst du es**
1. Ticket erstellen.  
2. Kategorie + Priorität setzen.  
3. Agent bearbeitet und löst.  
4. Kunde erhält Updates.

## 10) Cloud
**Was ist das?**
Dateien speichern und teilen.

**So machst du es**
1. Ordner anlegen.  
2. Datei hochladen.  
3. Freigabe an Benutzer/Gruppe.  
4. Link erstellen (wenn nötig).

## Fehler-Checkliste (wenn etwas nicht klappt)
1. Bist du eingeloggt?  
2. Pflichtfelder ausgefüllt?  
3. Richtige Rolle/Gruppe?  
4. Routing/Regeln geprüft?  
5. Seite neu laden oder Server neu starten.
