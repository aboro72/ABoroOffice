{% trans "Hilfe" %}

CloudService -> Dateien, Versionen und Freigaben.

Dateiverwaltung

  • Ordner erstellen, Dateien hochladen.
  • Versionen werden automatisch gespeichert.
  • Papierkorb für gelöschte Dateien.

Teilen

  • Freigaben an Benutzer/Gruppen.
  • Öffentliche Links (falls aktiviert).
  • Passwort/Expiry für Links (optional).

Plugin-Lifecycle

  1. Plugin installieren (ZIP/Repository).
  2. Aktivieren -> Widget erscheint.
  3. Optionen konfigurieren (falls vorhanden).
  4. Deaktivieren/Entfernen -> Widget verschwindet.

Typische Fehler

  • Upload fehlgeschlagen -> Dateigröße/Typ prüfen.
  • Plugin nicht sichtbar -> Aktivierung + Position prüfen.
  • Freigabe ohne Zugriff -> Rechte/Gruppe prüfen.
